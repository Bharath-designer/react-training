## Dynamic Items Manager - React Hooks

#### Concepts I used in this Item Manager React App

- React Hooks 
    - useEffect
    - useLayoutEffect
    - useContext
    - useReducer
    - useRef
- Context Provider (for managing state)
- Debouncing effect (for search)
- Custom Hooks (for filtering)
